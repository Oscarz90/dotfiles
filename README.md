# dotfiles

These are my dotfiles for ubuntu 18, just download, execute `./init.sh` and enjoy it.

Includes:

+ _zsh_ main configuration
+ _starship_ shell prompt
+ path prepend for my binaries
+ custom _functions_
+ custom _aliases_
+ custom _env variables_
+ .gitconfig and .gitignore_global files
